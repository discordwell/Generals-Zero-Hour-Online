# Source Save Fixtures

Place real C&C Generals or Zero Hour .sav files captured from the original C++ game in this directory.

These files are intentionally not generated by the browser port. The save-core parity report scans this directory only so generated TS saves cannot accidentally clear the wet C++ fixture blocker.

Recommended fixture names:
- `zh-skirmish-usa-open.sav`
- `zh-campaign-usa01-open.sav`
- `generals-skirmish-usa-open.sav`
